Rock O Rama, un site ayant pour but d'aider à se plonger dans l'univers du Rock et du Métal. Totalement débutant-friendly, il répertorie également différent forum sur lesquels les gens peuvent échanger librement.

Le site comprend : 
Une page d'accueil
Une page redirigeant vers la billeterie de concerts à venir
Une page réacpitulant les concerts passés
Une page résumant l'histoire du rock et du métal (grace à une vidéo)
Une page redirigeant vers des forums


Utilise :
- HTML5
- CSS3
- Git et Github


Choix techniques :
- Le CSS a été découpé en plusieurs fichiers, correspondant aux différents rôles ou les différentes pages du site.
- Aucune utilisation de div
- Design responsive grâce à des media queries
